## 🎯 What is AIPS?

**AIPS — AI Interview Preparation System** is a smart, end-to-end interview preparation platform that simulates the **complete real-world hiring process** — from company research all the way to the final HR round — powered by AI.

> *"Practice like it's real. Perform like it's easy."* 🚀

No two tests are ever the same. Every session generates a **unique mix** of expert-curated + AI-generated questions, so candidates always face fresh, relevant challenges.

---

## ✨ Key Features

### 🏢 Company & Domain Intelligence
- Enter any **company name** or **domain** (e.g. TCS, Infosys, Finance, IT)
- AIPS instantly shows the **complete hiring flow** for that company
- Previously searched companies are **saved** for quick access next time

### 📝 Smart Question Engine

<div align="center">

| Type | Share | Source |
|:---|:---:|:---|
| Pre-curated expert questions | **70%** | Hand-picked & verified |
| AI generated questions | **30%** | Freshly generated every session |
| **Total per test** | **100** | Unique every time 🎯 |

</div>

### 🔄 Complete Interview Round Simulation
📋 APTITUDE ROUND
├── 🔢 Quantitative Reasoning
├── 💬 Verbal Ability
└── 🧠 Logical Reasoning
💻 TECHNICAL ROUND
├── 👨‍💻 Coding Challenges
├── 📚 Subject Knowledge
└── 🛠️ Domain Specific Questions
🤝 HR ROUND
├── 🗣️ Behavioural Questions
├── 🎯 Situational Questions
└── 💼 Company Fit Assessment

### 📊 AI-Powered Performance Report
After every test, AIPS generates a **smart detailed report**:

- ✅ Total score & percentage
- 📈 Section-wise performance breakdown
- 🔴 Weak topics that need improvement
- 💡 Personalized "what to study next" suggestions
- 📉 Track progress across multiple attempts

## ⚙️ Getting Started

### Prerequisites
```bash
Node.js  >= 14
Python   >= 3.8
MySQL
```

### 1️⃣ Clone the repository
```bash
git clone https://github.com/khushikushwah-source/AIPS.git
cd AIPS
```

### 2️⃣ Backend setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 3️⃣ Frontend setup
```bash
cd frontend
npm install
npm start
```

### 4️⃣ Open in browser
http://localhost:3000
